Apatiency

I was part of a student group that developed an app to assist healthcare professionals in caring for Alzheimer's patients. It was a truly fascinating experience, as we were creating something with the potential to make a positive impact. This project was a collaboration between Hanzehogeschool Groningen and Dio Agency.

Color. ü§∑

Because this project was centered around Alzheimer's disease which is a serious condition we wanted the base of the colors to be stable and calm. This is why we took a blue palette with two lighter and two darker tints. We eventually decided to use this color for the text and not for the background.

The three main colors are a combination of orange, yellow and green. This was the perfect contrast to the main blue color. With this fresh look we had the perfect set-up for our app that interestingly had three main functions corrosponding with these colors.

Like I've mentioned, the main audience this project was healthcase workers with patients that suffer from Alzheimer's disease. A neutral and readable font was essential. This is why we choose Roboto Regular which is a font widely used because of it's natural feeling reading rhythm.

Creating a hierarchy using font-sizes was funnily enough not even that necessary due to the color hierarchy we created earlier.
